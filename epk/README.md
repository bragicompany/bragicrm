# EPK — Material de prensa de los artistas

Sube aquí el EPK (electronic press kit) en PDF de cada artista. Sugerencia de nombres:

- `dani.pdf` — EPK de Dani Vásquez
- `davikane.pdf` — EPK de Davikane

Claude lee estos PDFs para extraer bio, links y proponer ganchos, y rellenar
`artistas.py`. Los PDFs **no se suben a Git** (están en `.gitignore`) para no
inflar el repositorio; este README sí se versiona para que la carpeta exista.
