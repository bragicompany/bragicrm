# assets — Imágenes de la app

Sube aquí el **logo de Bragi** para que aparezca en la firma de los correos.

- Nombre sugerido: **`logo.png`**
- Formato: **PNG** (idealmente con fondo transparente) o JPG.
- Tamaño: que NO sea pesado. Ideal **ancho ~200–400 px** y **menos de ~100 KB**
  (un logo muy grande hace el correo pesado y puede caer en Promociones/Spam).

El logo se incrusta **dentro** del correo (viaje con el email, sin necesidad de
alojarlo en una web). Recordatorio: esto es el logo en el cuerpo/firma, NO el avatar
del remitente (eso es BIMI, aparte).
